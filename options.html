About Lightning Bug

Lightning Bug adds a few extra features to FireFly Learning that make it easier to set tasks for online lessons.

  • A button on each lesson in the Day and Week planner to create a task.
  • The button changes colour to show when a task has been set for that lesson.
  • The button links to the task definition allowing you to quickly check task completion and/or edit the task

Lightning Bug is in no way affiliated with FireFly Learning. If you find any problems or need help with Lightning Bug please send me a twitter DM to @eduwaller.
